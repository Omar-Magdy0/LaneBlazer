Detailed Description

template<uint8_t D1_pin, uint8_t D2_pin, uint8_t EN_pin, uint8_t D1_logic, uint8_t EN_logic>
struct _L293D_Drive_hal< D1_pin, D2_pin, EN_pin, D1_logic, EN_logic >

Initializes the motor driver pins and sets up PWM resolution and frequency.

This function configures the specified pins (D1_pin, D2_pin, EN_pin) as output pins. It also sets the PWM resolution and frequency for the motor driver.

Template Parameters
D1_pinThe pin number for the first direction control.
D2_pinThe pin number for the second direction control.
EN_pinThe pin number for the enable control.
D1_logicThe logic level for the first direction control.
EN_logicThe logic level for the enable control.
